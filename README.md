# Biblioteca Virtual - 4° Sem

Este projeto é um sistema de gerenciamento de biblioteca, projetado para simular o funcionamento de uma biblioteca real, permitindo cadastrar livros, registrar usuários, gerenciar empréstimos e gerar arquivos com os dados processados.

## Instruções de utilização
1. Clone o repositório através do seu terminal `https://github.com/gabrielfaia/ihc-4-pcg`;
2. Use `cd poo-4-biblioteca` para alterar o diretório;
3. Execute o arquivo principal com `python main.py`.
